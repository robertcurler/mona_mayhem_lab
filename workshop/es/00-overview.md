# 🎮 Mona Mayhem — Taller de GitHub Copilot en VS Code

---

> **Duración:** ~1 hora
> **Nivel:** Intermedio
> **Stack:** Astro / Node.js / TypeScript

Construye una app retro arcade de comparación de contribuciones de GitHub — **Mona Mayhem** — mientras aprendes todo el espectro de funciones de Copilot en VS Code: ingeniería de contexto, Plan Mode, Agent Mode, desarrollo orientado al diseño y flujos de trabajo multi-agente.

---

## 📋 Lista de Verificación Rápida

Antes de comenzar, verifica:

- [ ] VS Code **v1.107+** (sin actualizaciones pendientes)
- [ ] Sesión iniciada con **GitHub Copilot** (Pro, Business o Enterprise)
- [ ] Node.js **18+** instalado
- [ ] Git instalado y configurado
- [ ] Panel de Chat abierto y Agent listo

> 💡 **Consejo:** ¡Usa el DevContainer para un entorno preconfigurado!

---

## 🎯 Lo Que Aprenderás

| # | Habilidad | Descripción |
|---|-----------|-------------|
| 1 | **Context Engineering** | Enseñar a la IA sobre tu código con instrucciones |
| 2 | **Plan Mode** | Diseñar la arquitectura antes de programar |
| 3 | **Agent Mode** | Dejar que Copilot cree funcionalidades completas de forma autónoma |
| 4 | **Desarrollo orientado al diseño** | Transformar la interfaz con tematización guiada por IA |
| 5 | **Flujos de trabajo multi-agente** | Agentes en segundo plano y en la nube para trabajo en paralelo |

## 💡 Consejos Pro

1. **Mantén el navegador abierto** — Observa las actualizaciones en vivo mientras programas
2. **Haz commits frecuentemente** — Guarda los estados funcionales con frecuencia
3. **Usa los checkpoints** — Revierte cambios inesperados con Checkpoints y Undo del chat
4. **Itera los planes 2+ veces** — Los planes mejoran con cada pasada de refinamiento
