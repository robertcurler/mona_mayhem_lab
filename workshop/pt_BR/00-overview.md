# 🎮 Mona Mayhem — Workshop de GitHub Copilot no VS Code

---

> **Duração:** ~1 hora
> **Nível:** Intermediário
> **Stack:** Astro / Node.js / TypeScript

Construa um app retrô estilo arcade para comparação de contribuições no GitHub — **Mona Mayhem** — enquanto aprende todo o espectro de recursos do Copilot no VS Code: engenharia de contexto, Plan Mode, Agent Mode, desenvolvimento orientado a design e fluxos de trabalho com múltiplos agentes.

---

## 📋 Checklist Rápido

Antes de começar, verifique:

- [ ] VS Code **v1.107+** (sem atualizações pendentes)
- [ ] Conectado ao **GitHub Copilot** (Pro, Business ou Enterprise)
- [ ] Node.js **18+** instalado
- [ ] Git instalado e configurado
- [ ] Painel do Chat aberto e Agent pronto

> 💡 **Dica:** Use o DevContainer para um ambiente pré-configurado!

---

## 🎯 O Que Você Vai Aprender

| # | Habilidade | Descrição |
|---|------------|-----------|
| 1 | **Context Engineering** | Ensine a IA sobre sua base de código com instruções |
| 2 | **Plan Mode** | Projete a arquitetura antes de programar |
| 3 | **Agent Mode** | Deixe o Copilot criar funcionalidades inteiras de forma autônoma |
| 4 | **Desenvolvimento Orientado a Design** | Transforme a interface com temas guiados por IA |
| 5 | **Fluxos Multi-Agent** | Agentes em background e na nuvem para trabalho paralelo |

## 💡 Dicas Importantes

1. **Mantenha o navegador aberto** — Acompanhe as atualizações ao vivo enquanto programa
2. **Faça commits frequentes** — Salve estados funcionais regularmente
3. **Use checkpoints** — Reverta alterações inesperadas com Checkpoints e Undo no chat
4. **Itere nos planos 2+ vezes** — Os planos melhoram a cada rodada de refinamento
